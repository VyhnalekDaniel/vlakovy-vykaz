# Výkaz vozidel - Flask menu verze

Webová verze původní konzolové aplikace s menu 1-11.

## Lokální spuštění

```bash
pip install -r requirements.txt
python app.py
```

Otevřít: http://127.0.0.1:5000

## Render

Build Command:
```bash
pip install -r requirements.txt
```

Start Command:
```bash
gunicorn app:app
```
